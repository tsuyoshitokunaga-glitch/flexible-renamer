# Flexible Image Renamer

画像をリネームし、ZIPで一括ダウンロードするためのツールです。
ShopifyやECサイトの画像管理業務向けに、ドラッグ＆ドロップや手入力での並び替えに対応しています。

## 特徴

- **柔軟なリネーム設定**:
  - **先頭文字列 (Prefix)**: 任意の固定テキスト
  - **連番タイプ**: 数字 (001, 002...) または 英字 (A, B... / a, b...)
  - **最小桁数**: 数字の場合、指定した桁数（例: 3桁→001）でゼロ埋め
  - **末尾文字列 (Suffix)**: 任意の固定テキスト
- **並び替え機能**:
  - **ドラッグ＆ドロップ**: マウス操作で直感的に並び替え
  - **Manual Sort (手動No.)**: Shopifyのように番号を入力して並び替え
- **個別設定**:
  - 特定の画像だけ「手入力」モードに切り替え、ファイル名の一部を自由に指定可能
- **対応拡張子**:
  - .jpg, .jpeg, .png, .gif, .webp
- **推奨環境**:
  - Google Chrome, Edge, Firefox, Safari (最新版)
- **料金プラン**:

  - **Premium Plan**: 月額990円（税込）で無制限
  - ※ 現在はプロトタイプのため、決済処理はモック（シミュレーション）です。

## 使用方法

1.  `index.html` をブラウザ（Chrome推奨）で開きます。
2.  **リネーム設定**を入力します。
    - 連番タイプで英字（大文字/小文字）も選択可能です。
3.  画像を**ドロップゾーン**にドラッグ＆ドロップします。
4.  画像の順番を調整します。
    - リスト内の画像をドラッグして移動。
    - または、画像左側の数値入力欄に番号を入力して変更（入力後に自動で並び替わります）。
5.  必要に応じて、各画像のモードを「手入力」に変更し、名前を指定します。
6.  **「ZIP作成＆ダウンロード」** ボタンをクリックします。
