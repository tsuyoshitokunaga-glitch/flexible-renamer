# Flexible Image Renamer

画像リネームを一括で！ブラウザ完結・インストール不要の画像リネームツールです。
EC商品画像の整理を「3秒」で完了させ、面倒な連番リネームを一括自動化して業務時間を大幅に短縮します。
ShopifyやECサイトの画像管理業務向けに最適化されています。

## 🌟 主な特徴

### セキュアな処理
- **完全クライアントサイド処理**: 画像データはすべてブラウザ内で処理され、サーバーにアップロードされることは一切ありません
- 画像の内容、ファイル名、メタデータなど、いかなる画像関連情報もサーバーに保存されません
- プライバシーとセキュリティが最大限に保護されます

### 柔軟なリネーム設定
- **先頭文字列 (Prefix)**: 任意の固定テキスト（例: `IMG_`）
- **連番タイプ**: 
  - 数字 (1, 2, 3...)
  - 英字 大文字 (A, B, C...)
  - 英字 小文字 (a, b, c...)
- **開始番号**: 連番の開始位置を自由に設定可能（デフォルト: 1）
- **最小桁数**: 数字の場合、指定した桁数でゼロ埋め（例: 3桁→001, 002...）
- **末尾文字列 (Suffix)**: 任意の固定テキスト（例: `_v1`）

### 便利な並び替え機能
- **ドラッグ＆ドロップ**: マウス操作で直感的に画像の順番を変更
- **手入力による並び替え**: 画像左側の番号入力欄に数値を入力して順番を変更（入力後に自動で並び替わります）

### 個別設定対応
- 特定の画像だけ「手入力」モードに切り替えて、ファイル名の一部を自由に指定可能
- 手入力モードでも先頭・末尾文字列は自動で付与されます

### Web画像の直接ドロップ
- Webサイト上の画像を直接ドラッグ＆ドロップで追加可能
- ⚠️ **注意**: サイトのセキュリティ設定（CORS）によっては読み込めない場合があります
- エラーが出る場合は、一度パソコンに画像を保存してからアップロードしてください

### 高速処理
- ブラウザ内で完結するため、処理が高速
- 大量の画像も短時間で処理可能

## 🚀 使用方法

1. **リネーム設定を入力**
   - 先頭文字列、連番タイプ、開始番号、最小桁数、末尾文字列を設定します

2. **画像をドロップゾーンに追加**
   - 画像ファイルをドラッグ＆ドロップしてアップロード
   - または、Webサイト上の画像を直接ドラッグ＆ドロップ（CORS制限あり）
   - 画面上部の「画像リスト」セクションにもダウンロードボタンがあります

3. **画像の順番を調整**
   - 画像をドラッグして移動
   - または、画像左側の番号入力欄に数値を入力

4. **個別に名前を指定（オプション）**
   - 画像右側の「手入力」を選択して、個別のファイル名を入力

5. **ZIP作成＆ダウンロード**
   - 「ZIP作成＆ダウンロード」ボタンをクリック
   - 自動的にZIPファイルが生成され、ダウンロードされます
